## What are variables

You can define variables with the keywords `let` `const` `var`

### let

```js
let a = 5;
a = 6;
// works normally
```

### const

```js
const a = 5;
a = 6;
// error
```

### var
```js
const a = 5;
a = 6;
// ok
```
